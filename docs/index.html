DOMのスタイルを確認 / 変更する

この要素のスタイルを確認 / 変更する

要素を右クリックして「検証」を選択するとデベロッパーツールが開きます。
スタイルを確認し、文字を※※にしてみましょう。

上書きされたスタイルや無効なスタイルを確認する

この要素のスタイルを確認する

※※は上書きされて無効になったスタイルです。

※※はプロパティ名が間違っているのでwarningアイコンが表示されます。

プロパティをつけたり外したりする

この要素のスタイルを確認する

※※の横のチェックボックスをつけ外しすることで、※※が変化します。

クラスをつけたり外したりする

「:hover」のスタイルを確認しましょう。

「.is-disabled」クラスを付与するとボタンがdisalbedになります。

ショートハンドプロパティを展開する

この要素のスタイルを確認する

marginの横にある▼をクリックすると、ショートハンドプロパティが展開できます。

画面右側の歯車のマークをクリックし、「試験運用版」（英語の場合は「Experiments」）から「Enable new Font Editor tool within the Styles Pane.」にチェックを入れます。

右側の「AA」ボタンでフォントエディターを表示し、文字を変更してみましょう。
